# JSON-split-and-save-to-txt

## Опис

Цей скрипт `split_json.py` призначений для обробки великих експортів чатів Telegram у форматі JSON. Він розбиває файл `result.json` на менші частини, кожна з яких містить не більше 400 000 слів, що робить їх придатними для аналізу за допомогою інструменту NotebookLM від Google.

NotebookLM — це інструмент штучного інтелекту, який дозволяє аналізувати, підсумовувати та задавати питання до документів. Великі файли експортів чатів можуть перевищувати обмеження на розмір або кількість слів, тому розбиття на частини дозволяє ефективно використовувати NotebookLM для глибокого аналізу розмов.

## Функціональність

- Завантажує JSON-файл експорту чату (`result.json`)
- Розраховує кількість слів у кожному повідомленні
- Розбиває повідомлення на частини, де кожна частина не перевищує 400 000 слів
- Зберігає кожну частину у окремий файл `result_partXX.txt` у форматі JSON
- Зберігає заголовок експорту (метадані чату) у кожній частині

## Вимоги

- Python 3.x
- Файл `result.json` у кореневій директорії проекту (експорт чату Telegram)

## Використання

1. Переконайтеся, що файл `result.json` знаходиться у тій самій директорії, що й скрипт.
2. Запустіть скрипт:

   ```bash
   python split_json.py
   ```

3. Скрипт створить файли `result_part01.txt`, `result_part02.txt` тощо у тій самій директорії.
4. Кожну частину можна завантажити до NotebookLM для окремого аналізу.

## Налаштування

- `INPUT_FILE`: Ім'я вхідного файлу (за замовчуванням `result.json`)
- `MAX_WORDS`: Максимальна кількість слів на частину (за замовчуванням 400 000)
- `OUTPUT_DIR`: Директорія для вихідних файлів (за замовчуванням поточна директорія)

## Приклад виводу

Після запуску скрипт виведе інформацію про кількість повідомлень та частин:

```
Завантаження файлу...
Всього повідомлень: 150000
  Оброблено 10000/150000 повідомлень, частин: 2
  ...

Всього частин: 5
```

## Застосування з NotebookLM

NotebookLM дозволяє:
- Підсумовувати довгі розмови
- Знаходити ключові теми та патерни
- Відповідати на питання про зміст чату
- Створювати нотатки та інсайти

Завдяки розбиттю на частини, ви можете аналізувати великі чати поетапно або порівнювати різні періоди часу.
